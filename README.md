Zhihu Cleaner does not collect, store, transmit, or share any personal information or browsing data.

All settings are stored locally using Chrome's storage API and never leave the user's device.

This extension is only used to hide ads, videos, paid content, and recommendation modules on Zhihu pages.

No user data is sold, transferred, or used for any purpose unrelated to the extension's functionality.
